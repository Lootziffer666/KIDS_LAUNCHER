## 🧭 Was ist KIDS LAUNCHER?

Ein **kinderfreundlicher Controller-first Game Launcher** für Android. Keine Werbung, keine In-App-Käufe, keine versteckten Menüs.

### Prinzipien

- 🎯 **Controller First** — Jede Interaktion per Gamepad
- 👶 **Kid-Safe** — Keine externen Links, Stores oder Ads
- 🎨 **Visuell klar** — Große Icons, wenig Text
- 🔒 **Eltern-Gate** — Einstellungen nur mit PIN
